## Этап 2: Программная реализация и вычислительный эксперимент

### Цель работы

1. Реализовать базовую двумерную модель формирования планетной системы из газопылевого диска
2. Проверить численный метод на упрощённом сценарии с центральной звездой
3. Провести вычислительные эксперименты для самогравитирующей системы частиц с отталкиванием и трением
4. Подготовить визуализации, графики и выводы по результатам моделирования

---

### 1. Краткая постановка задачи

Формирование планетных систем рассматривается как эволюция вращающегося газопылевого диска, в котором локальные сгущения взаимодействуют гравитационно, сталкиваются и перераспределяют энергию.

Для каждой частицы с массой `m_i` и радиус-вектором `r_i` интегрируется система:

```text
m_i d²r_i / dt² = Fгр_i + Fотт_i + Fтр_i
```

В модели учитываются:

- ньютоновское гравитационное взаимодействие всех пар частиц
- короткодействующее отталкивание при сближении
- касательное трение, гасящее относительное движение поверхностей
- случайная инициализация частиц в диске радиуса `r0`
- кеплеровский профиль начальных скоростей `v ~ r^(-1/2)`

---

### 2. Программная реализация

Модель реализована в скрипте `planetary_stage2.py` с использованием `numpy` и `matplotlib`.

Для численного интегрирования использована схема **velocity Verlet**:

```python
pos_next = pos + vel * dt + 0.5 * acc * dt**2
vel_half = vel + 0.5 * acc * dt
acc_next = compute_forces(pos_next, vel_half, ...)
vel_next = vel + 0.5 * (acc + acc_next) * dt
```

**Что сохраняет программа:**

- траектории контрольной группы частиц
- снимки конфигурации системы в начале, середине и конце расчёта
- кинетическую, потенциальную и полную энергии
- среднеквадратичный радиус диска `R_rms`
- размер крупнейшего кластера частиц

**Параметры вычислительных экспериментов:**

| Сценарий | N | Δt | Шагов | k | β |
|---|---:|---:|---:|---:|---:|
| Центральная звезда | 48 | 0.01 | 900 | 0 | 0 |
| Самогравитация без трения | 96 | 0.0025 | 1800 | 0.0018 | 0 |
| Самогравитация с трением | 96 | 0.0025 | 1800 | 0.0018 | 0.7 |

### 3. Результаты вычислительного эксперимента

#### 3.1 Проверка схемы на модели с центральной звездой

Первый сценарий воспроизводит движение частиц в центральном поле без их взаимного притяжения.

**Получено:**

- орбиты остаются ограниченными
- разлёт частиц не наблюдается
- инициализация кеплеровских скоростей работает корректно
- при отсутствии трения полная энергия почти не дрейфует

#### 3.2 Самогравитирующий диск без трения и с трением

Во втором сценарии неподвижная центральная точка убирается, частицы взаимодействуют друг с другом гравитационно, а при сближении включаются силы отталкивания и трения.

**Наблюдения:**

- без трения диск дольше остаётся динамически горячим и формирует только неустойчивые сгущения
- с трением касательные относительные скорости гасятся
- вещество быстрее собирается в компактные группы
- трение заметно ускоряет кластеризацию даже в базовой модели

#### 3.3 Количественная оценка агрегации

Для анализа использовались две метрики:

1. среднеквадратичный радиус диска `R_rms`
2. размер крупнейшего кластера частиц

К концу расчёта крупнейший кластер:

- без трения достигает примерно **31 частицы**
- с трением достигает примерно **50 частиц**

Это подтверждает, что диссипация механической энергии ускоряет уплотнение вещества.
