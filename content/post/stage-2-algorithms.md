## Этап 2: Алгоритмы

### Краткое содержание

**Цель этапа:** Разработка и исследование алгоритмов численного решения задачи гравитационного взаимодействия N тел для моделирования процесса образования планетной системы.

**Основные алгоритмы:**
- **Алгоритм Верле (Verlet)** — метод численного интегрирования уравнений движения, обеспечивающий сохранение энергии в долгосрочной перспективе
- **Прямой метод (Direct Summation)** — попарное вычисление гравитационных сил между всеми частицами, сложность O(N²)
- **Алгоритм Барнса-Хата (Barnes-Hut)** — иерархический алгоритм с использованием октодерева, сложность O(N log N)
- **Быстрый метод мультиполей (FMM)** — алгоритм с линейной сложностью O(N) для больших систем

**Численная схема:**
- Интегрирование уравнений движения частиц с учётом гравитационных сил
- Обработка столкновений и объединения частиц
- Контроль сохранения полной энергии и момента импульса системы
