## Этап 1: Модель

### Краткое содержание

**Научная проблема:** Образование планетных систем — одна из фундаментальных проблем современной астрофизики. Понимание механизмов формирования планет имеет ключевое значение для объяснения строения нашей Солнечной системы и интерпретации наблюдений экзопланетных систем.

**Теоретические основы:**
- Согласно небулярной гипотезе, Солнечная система образовалась из вращающегося газопылевого облака примерно 4,5 млрд лет назад
- При гравитационном сжатии облако становилось плоским диском (протопланетный диск)
- На периферии диска отделялись кольца вещества, из которых впоследствии сформировались планеты
- Процесс сопровождался гравитационной неустойчивостью и фрагментацией облака

**Математическая модель:**
- Построена модель формирования планетной системы на основе гравитационного взаимодействия N тел
- Используется система дифференциальных уравнений движения частиц в поле гравитационных сил
- Применяется алгоритм Верле для численного интегрирования уравнений динамики
- Учитываются столкновения и слипание частиц, диссипативные процессы
- Изучена вычислительная сложность алгоритма: O(N²) для прямого метода
