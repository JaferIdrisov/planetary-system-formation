## Этап 1: Модель

### Цель работы

1. Изучить научную проблему образования планетных систем
2. Описать теоретические основы задачи
3. Построить математическую модель формирования планетной системы из газопылевого облака на основе гравитационного взаимодействия частиц

---

### 1. Научная проблема

**Происхождение планетных систем** — одна из фундаментальных проблем астрофизики:

- Связана с пониманием строения Солнечной системы
- Актуальна для интерпретации наблюдений экзопланетных систем, число которых стремительно растёт благодаря современным телескопам
- Требует численного моделирования сложной задачи многих тел
- Необходимо учитывать гравитационное взаимодействие, столкновения и слипание частиц, а также диссипативные процессы

---

### 2. Теоретическое описание задачи

#### 2.1 Большой взрыв и формирование структуры Вселенной

- Вселенная возникла из точки в результате Большого взрыва примерно **13,7 млрд лет назад**
- Первоначально слабые нерегулярности в распределении вещества усиливались из-за **гравитационной неустойчивости**
- Распад уплотнений на отдельные сгущения дал начало **протогалактикам**
- Фрагментация протогалактических облаков вела к возникновению **первых галактик** и формированию звёзд
- Наиболее массивные звёзды исчерпали топливо за 3–8 млн лет и превратились в **сверхновые**, синтезируя тяжёлые элементы
- Звёзды солнечной массы после выгорания водорода (порядка 10 млрд лет) превращаются в **красные гиганты**

#### 2.2 Образование Солнечной системы. Небулярная гипотеза

Газопылевое облако, из которого образовались Солнце и планеты, имело **ненулевой момент импульса** (вращалось).

**Процесс формирования:**

1. При гравитационном сжатии расстояние от оси вращения сокращалось → скорость вращения увеличивалась (закон сохранения момента импульса)
2. В плоскости, перпендикулярной оси вращения, сжатие происходило медленнее → облако становилось плоским
3. Из-за гравитационной неустойчивости на периферии диска отделялись **кольца вещества**
4. Кольца сгустились в планеты

**Закон Тициуса–Боде:**

Расстояние планет от Солнца описывается формулой:

```
r_n = 0,3 × 2^n + 0,4 а.е.
```

где за 1 а.е. принимается расстояние Земля–Солнце, n = −∞ для Меркурия, n = 3 для пояса астероидов. Выполняется с точностью ~5%.

### 3. Математическая модель

#### 3.1 Постановка задачи

Построить модель формирования планетной системы на основе гравитационного взаимодействия N частиц, учитывающую:

- Гравитационное притяжение между частицами
- Силы отталкивания при сближении
- Силы трения
- Слипание частиц при столкновении

#### 3.2 Уравнения динамики частиц

Движение каждой частицы описывается уравнением Ньютона:

```
m_i × a_i = Σ F_ij
```

где суммирование ведётся по всем частицам, с которыми взаимодействует i-я частица.

**Силы взаимодействия:**

##### 3.2.1 Гравитационная сила

```
F_g = G × m_i × m_j / r²
```

где G — гравитационная постоянная, r — расстояние между частицами.

##### 3.2.2 Сила отталкивания

При малых расстояниях действует сила отталкивания:

```
F_r(b) = k × (R_i + R_j - b)^8
```

где b — расстояние между центрами частиц, k — коэффициент жёсткости. Восьмая степень обеспечивает быстрое нарастание силы при уменьшении расстояния.

##### 3.2.3 Сила трения

Сила трения перпендикулярна радиус-вектору взаимодействия и направлена против движения:

```
F_f = β × W⊥ × F_r(b) × n
```

где β — коэффициент трения, W⊥ — относительная скорость поверхностей частиц перпендикулярно радиусу, n — единичный вектор вдоль силы трения.

Для двумерной модели:

```
n = (−b_y, b_x) / √(b_x² + b_y²)
```

##### 3.2.4 Слипание частиц

При полном слипании двух частиц образуется одна частица:

**Радиус:**
```
R = ∛(R_i³ + R_j³)
```

**Масса:**
```
m = m_i + m_j
```

**Координаты и скорость** (по закону сохранения импульса):
```
r = (m_i × r_i + m_j × r_j) / (m_i + m_j)
v = (m_i × v_i + m_j × v_j) / (m_i + m_j)
```

#### 3.3 Алгоритм Верле

Для численного интегрирования уравнений движения используется **алгоритм Верле**:

```
r(t + Δt) = 2r(t) − r(t − Δt) + a(t) × Δt²
```

**Преимущества алгоритма:**

- Второй порядок точности
- Хорошее сохранение энергии
- Обратимость по времени

#### 3.4 Вычислительная сложность

Число операций для вычисления взаимодействий каждой частицы с каждой возрастает как **O(N²)**.

Существуют способы ускорения:

- Моделирование «частица–сетка»
- Мультипольное разложение
- Алгоритм Барнса–Хата: **O(N log N)**
- Быстрый метод мультиполей (FMM): **O(N)**
