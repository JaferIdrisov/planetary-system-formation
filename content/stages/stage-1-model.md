## Этап 1: Модель

### Краткое содержание

**Научная проблема:** Образование планетных систем — одна из фундаментальных проблем астрофизики, связанная с пониманием механизмов формирования планет из газопылевого облака.

**Теоретические основы:**
- Согласно небулярной гипотезе, Солнечная система образовалась из вращающегося газопылевого облака
- При гравитационном сжатии облако становилось плоским диском
- На периферии диска отделялись кольца вещества, из которых сформировались планеты

**Математическая модель:**
- Построена модель формирования планетной системы на основе гравитационного взаимодействия частиц
- Используется задача многих тел с учётом гравитационных сил
- Применяется алгоритм Верле для численного интегрирования уравнений динамики
- Изучена вычислительная сложность алгоритма
